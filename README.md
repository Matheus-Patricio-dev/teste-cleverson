# Gerenciamento de Usuários – CRUD Simples

Aplicação web simples para gerenciamento de usuários, permitindo listar, cadastrar, editar e excluir registros.  
O projeto foi desenvolvido como teste técnico utilizando apenas tecnologias básicas, com foco em clareza, organização e funcionamento.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PHP puro
- JSON como banco de dados

---

## Como rodar o projeto

### Pré-requisitos
- PHP 7 ou superior instalado

Verifique a instalação do PHP:
```bash
php -v
```

### Clone o Projeto
```bash
git clone https://github.com/Matheus-Patricio-dev/teste-cleverson
```

### Inicie o servidor PHP 
```bash
php -S localhost:8080
```

### Acesse o localhost
http://localhost:8000/index.php
